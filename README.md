# AiNexus

## 📖 Description

AiNexus — это AI-driven платформа для проведения онлайн-собеседований абитуриентов с использованием автоматизированного интервью и прокторинга.

Система позволяет заменить традиционные анкеты и эссе на интерактивное AI-интервью, обеспечивая более объективную и масштабируемую оценку кандидатов.

---

InVision U — инновационный университет с 100% грантами для подготовки будущих лидеров.

Однако существующий процесс отбора сталкивается с рядом ограничений:

- Текущая система оценивает в первую очередь заявку, а не личность кандидата  
- Кандидаты с высоким потенциалом могут быть недооценены из-за слабой самопрезентации  
- Текстовые эссе становятся менее надёжным инструментом оценки в эпоху генеративного ИИ  
- Ручной скрининг ограничен по масштабируемости и глубине анализа  

---

## 💡 Solution

AiNexus предлагает новый подход к отбору кандидатов:

- 🤖 AI-интервью вместо статичных форм
- 🧠 Оценка мышления и потенциала кандидата
- 🎥 Прокторинг с проверкой личности
- ⚡ Масштабируемая автоматизация отбора

## 🏗 Architecture

Система построена на микросервисной архитектуре с взаимодействием через REST API:

- **Frontend (Next.js)** — интерфейс пользователя
- **Main Service (AiNexus)** — бизнес-логика
- **Email Service** — отправка ссылок и уведомлений
- **Face Verification Service (FastAPI)** — проверка личности
- **AI Service (Flowise)** — проведение интервью

Все сервисы взаимодействуют через HTTP API и используют PostgreSQL как основное хранилище.

## 🔄 User Flow

1. Пользователь подаёт заявку  
2. Получает email с временной ссылкой  
3. Переходит по ссылке → открывается AI-интервью (чат)  
4. Проходит собеседование  
5. Система:
   - задаёт вопросы  
   - анализирует ответы  
   - формирует score  
6. Администратор:
   - просматривает результаты  
   - принимает финальное решение
